BIOS

  • Basic Input Output System - 1975
  • Software stored on a small chip on the motherboard - ROM - flash memory
  • Rewritable ( flash memory )
  • First software run when computer power on

BIOS steps

  • Initialize and test the system hardware
  • Load a bootloader or an operating system
  • Runtime service for operating system - keyboard, display, input/output device , etc. but it don't use with linux, windows and ...
  • Config hardware - change boot order - view fan speed - etc
UEFI
  • Unified Extensible Firmware Interface
  • Replace with traditional BIOS - 2014
  • Software interface between an operating system and firmware
  • Initializing/Booting computer before passing to operting system

Advantages
  • Faster boot
  • Runtime service - available for os loader or os
  • Firmware validation method - which boot code is secure
  • Better interface
  • Mouse, Network, ...
  • Boot from GPT
  • CMS - compability Support Module - bios-based computer(os installed)
Secure boot

  • UEFI protocol
  • Secure "boot code"
  • Validate program with it's own key
  • Prevent malware from launching at boot time

Secure boot or restricted boot ?

  • enable/disable secure boot.
  • run prgoram's that doesn't have certificate key.
  • can't install os that doesn't have certificate key.
fsf statement
    We, the undersigned, urge all computer makers implementing UEFI's so-called "Secure Boot" to do it in a way that allows free software operating systems to be installed. To respect user freedom and truly protect user security, manufacturers must either allow computer owners to disable the boot restrictions, or provide a sure-fire way for them to install and run a free software operating system of their choice. We commit that we will neither purchase nor recommend computers that strip users of this critical freedom, and we will actively urge people in our communities to avoid such jailed systems.

MBR
  • Master Boot Record
  • Boot sector at fist sector of mass storage - preceding the first partition
  • Contain executable code to run operating system or passing to loader's second stage like grub

MBR Content
  • Partition table
  • Bootstarp code - passing to bootable partition loader, ...

MBR information
  • Number of partition - 4 primary
  • Size of partition ( 2 TB - max )
